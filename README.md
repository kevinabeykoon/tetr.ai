# Tetr.ai Retro Game



# Built With
1. Python
2. MediaPipe framework from Google Developers
3. pygame

## Getting Started
### Dependencies
The following libraries must be downloaded within the scope of the project:

For the hand_gesture_recognition:
* pip install opencv-python
* pip install mediapipe

For the Tetris game:
* pip install pygame
